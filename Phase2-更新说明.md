## 🔧 GitHub配置（可选）

如果想自动推送到GitHub：

1. 前往GitHub创建Personal Access Token
   - Settings → Developer settings → Personal access tokens → Tokens (classic)
   - 勾选 `repo` 权限
   - 生成Token并复制

2. 在系统设置页面配置GitHub Token

3. Demo生成后会自动推送到GitHub

**不配置GitHub也可以**：Demo代码会保存在系统中，可以直接下载ZIP包。

## 📞 遇到问题？

常见问题：

**Q: Demo生成失败怎么办？**  
A: 检查API配置是否正确，确保有足够的Token额度。

**Q: 无法推送到GitHub？**  
A: 确认已在设置页面配置GitHub Token，并检查Token权限。

**Q: 生成的代码能直接运行吗？**  
A: 需要在微信开发者工具中导入，并按照README配置云开发环境。

**Q: 可以重新生成Demo吗？**  
A: 可以，在PRD页面点击"重新生成"即可。
